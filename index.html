About This Project

This is a starting point for my Digital Garden.
Here, I aim to find a space for self-expression through writing, UX design, and aesthetically pleasing code.

This will never be finished, but is rather meant to be an organically evolving outlet for my thoughts.
Thus, I hereby present it as is right now - no more, no less.

The Garden is not structured to be read in a linear fashion, but rather explored by a curious mind.
Most pages can be reached from the Navigation Tree to the left, but some are loose leaves blowing in the wind, only found via hyperlinks from other pages.
